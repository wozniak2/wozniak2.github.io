Rozkład dwumianowy

Rozkład dwumianowy

to dyskretny rozkład prawdopodobieństwa opisujący liczbę sukcesów w ciągu niezależnych prób, z których każda ma stałe prawdopodobieństwo sukcesu.

Przykłady:

  • rzut monetą
  • rzut kostką
  • losowanie z talii kart

Rozkład dwumianowy w R

Cztery dostępne funkcje:

  • funkcja gestosci prawdopodobienstwa dbinom() - okresla prawdopodobienstwo, ze zmienna losowa przyjmie daną wartość x

Rozkład dwumianowy w R

Cztery dostępne funkcje:

  • funkcja gestosci prawdopodobienstwa dbinom() - okresla prawdopodobienstwo, ze zmienna losowa przyjmie daną wartość x
  • dystrybuanta pbinom() - określa prawdopodobieństwo tego, ze dana zmienna losowa przyjmie wartość równą bądź mniejszą x

Rozkład dwumianowy w R

Cztery dostępne funkcje:

  • funkcja gestosci prawdopodobienstwa dbinom() - okresla prawdopodobienstwo, ze zmienna losowa przyjmie daną wartość x
  • dystrybuanta pbinom() - określa prawdopodobieństwo tego, ze dana zmienna losowa przyjmie wartość równą bądź mniejszą x
  • generator liczb losowych rbinom()
  • funkcja kwantylowa - odwrotna dystrybuanta qbinom() - przypisuje konkretne wartości dla danego obszaru (sumy) prawdopodobieństwa

Z wykorzystaniem for rzucamy 20 razy dwoma kostkami jednocześnie i zapisujemy wyniki eksperymentu.

# tworzymy puste wektory
kostka1 <- NULL
kostka2 <- NULL

# rzucamy kostkami 20 razy - funkcje *rzuc.koscia* opracowaliśmy wcześniej 
for (i in 1:20) {
  kostka1[i] <- rzuc.koscia(1)
  kostka2[i] <- rzuc.koscia(1)
  }

# ile razy liczba oczek na 1 kostce = 2
sum(kostka1==kostka2)
## [1] 1
